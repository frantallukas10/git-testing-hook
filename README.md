# test-hook

> Git pre-commit hook that just runs your tests. :green_apple:

## Install

[Yarn](https://github.com/yarnpkg/)

```sh
yarn add test-hook --dev
```

[NPM](https://www.npmjs.com/)

```sh
npm install test-hook --save-dev
```
